HTML

html은 HTML(하이퍼텍스트 마크업 언어, HyperText Markup Language)는 웹페이지를 만들 때 사용하는 기본적인 언어예요. 웹사이트의 구조를 정의하는 역할을 하며, 텍스트, 이미지, 링크, 버튼 등 다양한 요소를 포함할 수 있어요.



CSS

CSS는 CSS(Cascading Style Sheets, 캐스케이딩 스타일 시트)는 HTML로 만든 웹페이지의 디자인과 스타일을 담당하는 언어예요.

JAVASCRIPT

javascript는 **JavaScript(JS)**는 웹페이지를 더 동적이고 인터랙티브하게 만들기 위해 사용하는 프로그래밍 언어예요. HTML이 웹페이지의 뼈대, CSS가 디자인, JavaScript는 기능과 동작을 담당해요!
